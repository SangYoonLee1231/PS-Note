## 나누어 입력받기

- 다음 입력값이 주어졌을 때 각각의 변수에 값을 저장하는 방법

```
1 2
```

```python
a, b  = map(int, input().split())

c = list(map(int, input(),split()))
# c에 두 입력값을 리스트(list)로 저장
```
